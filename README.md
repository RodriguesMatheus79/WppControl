# Controle de Leads

Aplicacao pessoal e local para controlar leads, estado da conversa e historico de abordagens.

## Como usar

Abra `index.html` no navegador. Os dados ficam salvos no proprio navegador via `localStorage`.

## Funcionalidades

- Cadastro de nome, telefone, origem, valor potencial, etapa e proximo contato.
- Etapas prontas: primeiro contato, oferta enviada, oferta desconto, follow-up, fechado e perdido.
- Historico simples de abordagens por lead.
- Busca, filtros por etapa e ordenacao.
- Indicadores de total, abertos, contatos para hoje e conversao.
- Exportacao e importacao em JSON para backup.
